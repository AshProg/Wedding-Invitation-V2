# Assets Folder

Place your files here:

## Canva Design Files
1. Export your Canva design as PNG or JPG
2. Name them sequentially: `canva-page1.png`, `canva-page2.png`, etc.
3. Or use any names you prefer and update the filenames in `index.html`

## Music File
1. Add your wedding song MP3 here
2. Name it `wedding-song.mp3`
3. Or use a different name and update the filename in `index.html`

## Supported File Types
- Images: PNG, JPG, JPEG, WebP
- Audio: MP3, OGG, WAV (MP3 recommended for best compatibility)
